**1. Create DocumentService.kt** in service package. Constructor-inject: `DocumentRepository`, `DocumentVersionRepository`, `DocumentApplicationLinkRepository`, `StorageService`. Add `private val tika = org.apache.tika.Tika()`. All public methods take `userId: UUID` as first parameter for user isolation.

**Methods to implement:**

**createDocument(userId, file: MultipartFile, title: String, category: DocumentCategory, description: String?):**
- Validate file type: call `tika.detect(file.bytes)` (use `file.bytes` to avoid InputStream consumption issues -- the bytes are read once and reused). Allowed MIME types: `application/pdf` and `application/vnd.openxmlformats-officedocument.wordprocessingml.document`. If not allowed, throw `InvalidFileTypeException("File type '$detectedType' not allowed. Allowed: PDF, DOCX")`.
- Create `DocumentEntity` with userId, title, category, description. Save via repository.
- Create `DocumentVersionEntity` with documentId = savedDoc.id!!, versionNumber = 1, storageKey = `"documents/${savedDoc.id}/${UUID.randomUUID()}.${getExtension(file.originalFilename)}"`, originalFilename = file.originalFilename ?: "unknown", contentType = detectedType, fileSize = file.size, note = null, isCurrent = true. Save via repository.
- Upload to S3: `storageService.upload(version.storageKey, file.bytes.inputStream(), file.size, detectedType)`
- Return `toDocumentResponse(savedDoc, savedVersion)`

**getDocument(userId, documentId):**
- Find by `findByIdAndUserIdAndArchivedFalse(documentId, userId)` or throw `NotFoundException("Document not found")`
- Load current version via `findByDocumentIdAndIsCurrent(documentId, true)`
- Return `toDocumentResponse(doc, currentVersion)`

**listDocuments(userId, category: DocumentCategory?, search: String?, pageable: Pageable):**
- Call `documentRepository.findAllByFilters(userId, category, search, pageable)`
- For each document, batch-load current versions: get all document IDs from page, then for each load current version. Map to responses.
- Return `Page<DocumentResponse>`

**updateDocument(userId, documentId, request: DocumentUpdateRequest):**
- Find document or throw NotFoundException
- Update title, description, category, updatedAt = Instant.now()
- Save and return response

**archiveDocument(userId, documentId):**
- Find document or throw NotFoundException
- Set archived = true, archivedAt = Instant.now(), updatedAt = Instant.now()
- Save. Return Unit.

**createVersion(userId, documentId, file: MultipartFile, note: String?):**
- Find document or throw NotFoundException
- Validate file type (same as createDocument)
- Get next version number: `documentVersionRepository.countByDocumentId(documentId) + 1`
- Clear current flag on all existing versions: `documentVersionRepository.clearCurrentFlag(documentId)`
- Create new DocumentVersionEntity with isCurrent = true, upload to S3
- Update document's updatedAt
- Return DocumentVersionResponse

**listVersions(userId, documentId):**
- Find document or throw NotFoundException
- Return `documentVersionRepository.findByDocumentIdOrderByVersionNumberDesc(documentId)` mapped to DTOs

**setCurrentVersion(userId, documentId, versionId):**
- Find document or throw NotFoundException
- Find version by `findByIdAndDocumentId(versionId, documentId)` or throw NotFoundException
- Clear current flag, set this version's isCurrent = true, save
- Update document's updatedAt
- Return DocumentVersionResponse

**downloadVersion(userId, documentId, versionId):**
- Find document or throw NotFoundException
- Find version or throw NotFoundException
- Call `storageService.download(version.storageKey)`
- Return Triple(inputStream, version.contentType, version.originalFilename)

**deleteVersion(userId, documentId, versionId):**
- Find document or throw NotFoundException
- Find version or throw NotFoundException
- If only 1 version exists, throw `ConflictException("Cannot delete the only version. Archive the document instead.")`
- Delete from S3: `storageService.delete(version.storageKey)`
- If deleted version was current, set the latest remaining version as current
- Delete version entity
- Update document's updatedAt

**linkDocumentToApplication(userId, request: LinkDocumentRequest):**
- Verify version exists and belongs to a document owned by this user
- Verify application exists and belongs to this user (query ApplicationRepository or use a simple `existsByIdAndUserId` -- add to ApplicationRepository if not there, or just check via documentVersionRepository and applicationRepository)
- Check not already linked: `existsByDocumentVersionIdAndApplicationId`; if exists throw `ConflictException("Already linked")`
- Save DocumentApplicationLinkEntity
- Return DocumentApplicationLinkResponse

**unlinkDocument(userId, documentVersionId, applicationId):**
- Verify ownership
- Delete link via `deleteByDocumentVersionIdAndApplicationId`

**getLinksForApplication(userId, applicationId):**
- Get links via `findByApplicationId(applicationId)`
- For each link, check if the version still exists; if not, set versionRemoved = true
- Return list of DocumentApplicationLinkResponse

**Helper: toDocumentResponse(entity, currentVersion)** and **toVersionResponse(entity)** — straightforward field mapping.

**Helper: getExtension(filename: String?)** — extract extension from filename, default to "bin" if none.

Mark the service with `@Service` and `@Transactional` on methods that write (or class-level `@Transactional` with `readOnly = true` default and override on write methods).

**2. Create DocumentController.kt** in controller package:
```
@RestController
@RequestMapping("/api/documents")
class DocumentController(private val documentService: DocumentService)
```

**Endpoints:**

- `POST /api/documents` (consumes MULTIPART_FORM_DATA_VALUE) — `@RequestPart("file") file: MultipartFile`, `@RequestPart("title") title: String`, `@RequestPart("category") category: String`, `@RequestPart("description", required = false) description: String?`. Parse category to `DocumentCategory.valueOf(category.uppercase())`. Return `ResponseEntity.status(HttpStatus.CREATED).body(result)`.

- `GET /api/documents` — `@RequestParam category: DocumentCategory?`, `@RequestParam search: String?`, `pageable: Pageable`. Return 200 with page.

- `GET /api/documents/{id}` — Return 200 with document response.

- `PUT /api/documents/{id}` — `@Valid @RequestBody request: DocumentUpdateRequest`. Return 200.

- `DELETE /api/documents/{id}` — Archive (soft delete). Return 204 No Content.

- `POST /api/documents/{id}/versions` (consumes MULTIPART_FORM_DATA_VALUE) — `@RequestPart("file") file`, `@RequestPart("note", required = false) note: String?`. Return 201.

- `GET /api/documents/{id}/versions` — List versions. Return 200.

- `PUT /api/documents/{id}/versions/{versionId}/current` — Set version as current. Return 200.

- `GET /api/documents/{id}/versions/{versionId}/download` — Download. Return `ResponseEntity<InputStreamResource>` with Content-Type and Content-Disposition headers: `header(HttpHeaders.CONTENT_DISPOSITION, "attachment; filename=\"$filename\"")`.

- `DELETE /api/documents/{id}/versions/{versionId}` — Delete version. Return 204.

- `POST /api/documents/links` — `@Valid @RequestBody request: LinkDocumentRequest`. Return 201.

- `DELETE /api/documents/links?documentVersionId={}&applicationId={}` — Unlink. Return 204.

- `GET /api/documents/links/application/{applicationId}` — Get links for application. Return 200.

All endpoints use `SecurityContextUtil.getCurrentUserId()` to get the authenticated user's ID. Wrap all path UUIDs with `@PathVariable`.
