**A. Create `frontend/components/documents/document-upload.tsx`**:

Drag-and-drop upload zone using react-dropzone.
1. Uses `useDropzone` hook from `react-dropzone`
2. Accept types: `{ "application/pdf": [".pdf"], "application/vnd.openxmlformats-officedocument.wordprocessingml.document": [".docx"] }`
3. `onDrop` callback: for each file, create FormData with fields:
   - `file`: the File object
   - `title`: file name without extension (`file.name.replace(/\.[^.]+$/, "")`)
   - `category`: "OTHER" (default)
   Call `useUploadDocument().mutate(formData)`. On success, toast "Document uploaded". On error, toast "Upload failed. Check the file format and try again."
4. Render: dashed border container (`border-2 border-dashed rounded-lg p-8 text-center cursor-pointer transition-colors`)
   - When drag active: `border-primary bg-primary/5`, text "Drop files here"
   - When idle: `border-muted-foreground/25`, text "Drag and drop files here, or click to browse"
   - Hint below: "PDF and DOCX files only" in `text-xs text-muted-foreground`
5. Show upload progress: when mutation is pending, show spinner icon next to file name

**B. Create `frontend/components/documents/document-list.tsx`**:

File browser table using shared DataTable.
1. Props: `documents: DocumentResponse[]`, `isLoading: boolean`, `onDelete: (doc: DocumentResponse) => void`

   - **Title**: accessor `title`
   - **File Name**: accessor `fileName`, `text-xs text-muted-foreground`
   - **Category**: accessor `category`, formatted (CV, COVER_LETTER -> "Cover Letter", PORTFOLIO, OTHER)
   - **Size**: accessor `fileSize`, formatted as KB/MB (e.g., "245 KB", "1.2 MB")
   - **Version**: accessor `versionNumber`
   - **Uploaded**: accessor `createdAt`, formatted date
   - **Actions**: Download button (anchor link to `API_BASE/documents/${id}/download` with `credentials: include` -- use a button that triggers `window.open(url)` or creates a hidden link) and Delete button
3. Loading: skeleton table rows
