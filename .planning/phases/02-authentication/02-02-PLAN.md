**JwtAuthenticationFilter.kt** (in security/ package):
- `@Component` class extending `OncePerRequestFilter`
- Constructor-inject: `JwtTokenProvider`, `UserDetailsService`, `TokenBlocklistRepository`
- `override fun doFilterInternal(request, response, filterChain)`:
  1. Extract token from `Authorization` header (strip "Bearer " prefix)
  2. If token is null or not valid, pass through to filterChain
  3. Get tokenId, check `tokenBlocklistRepository.existsByTokenId(tokenId)` -- if blocklisted, pass through

  5. Get username via `jwtTokenProvider.getUsername(token)`
  6. Load user via `userDetailsService.loadUserByUsername(username)`
  7. Create `UsernamePasswordAuthenticationToken(userDetails, null, userDetails.authorities)`
  8. Set on `SecurityContextHolder.getContext().authentication`
  9. Continue filter chain
- `private fun extractToken(request: HttpServletRequest): String?` -- get "Authorization" header, check startsWith "Bearer ", substring(7)

**AuthService.kt** (in service/ package):
- `@Service` class, constructor-inject: `UserRepository`, `TokenBlocklistRepository`, `EmailVerificationTokenRepository`, `PasswordEncoder`, `JwtTokenProvider`
- `fun register(request: RegisterRequest): MessageResponse`:
  1. Check `userRepository.existsByEmail(request.email)` -- if true, throw `RegistrationException("Registration failed")` (generic message prevents email enumeration)
  2. Create `UserEntity(email = request.email, password = passwordEncoder.encode(request.password), enabled = false)` -- enabled=false until email verified
  3. Save user via `userRepository.save(user)`
  4. Create email verification token: `UUID.randomUUID().toString()`, save as `EmailVerificationToken(user = user, token = token, expiresAt = Instant.now().plus(24, ChronoUnit.HOURS))`
  5. Log verification link to console: `logger.info("Email verification link: http://localhost:8080/api/auth/verify?token=$token")` (per CONTEXT.md decision: log to console in local dev)
  6. Return `MessageResponse("Registration successful. Please check your email to verify your account.")`

- `fun login(request: AuthRequest): Pair<AuthResponse, String>` (returns AuthResponse + refresh token string):
  1. `val user = userRepository.findByEmail(request.email) ?: throw AuthenticationException("Invalid credentials")`
  2. `if (!passwordEncoder.matches(request.password, user.password)) throw AuthenticationException("Invalid credentials")` -- same error for both cases
  3. `if (!user.enabled) throw AuthenticationException("Account not verified. Please check your email.")`
  4. Create access token: `jwtTokenProvider.createAccessToken(user.id!!, user.email, user.role.name)`
  5. Create refresh token: `jwtTokenProvider.createRefreshToken(user.id!!, user.email)`
  6. Return `Pair(AuthResponse(accessToken, jwtTokenProvider.getAccessExpirationMs() / 1000), refreshToken)`

- `fun refresh(refreshToken: String): Pair<AuthResponse, String>`:
  1. `if (!jwtTokenProvider.validateToken(refreshToken)) throw AuthenticationException("Invalid refresh token")`
  2. `if (jwtTokenProvider.getTokenType(refreshToken) != "refresh") throw AuthenticationException("Invalid token type")`
  3. Check blocklist: `if (tokenBlocklistRepository.existsByTokenId(jwtTokenProvider.getTokenId(refreshToken))) throw AuthenticationException("Token revoked")`
  4. Blocklist the old refresh token: save `TokenBlocklistEntry(tokenId = jwtTokenProvider.getTokenId(refreshToken), expiresAt = jwtTokenProvider.getExpiration(refreshToken).toInstant())`
  5. Get username, look up user, create new access + refresh tokens (same as login)
  6. Return new Pair

- `fun logout(accessToken: String, refreshToken: String?)`:
  1. If accessToken is valid, add to blocklist: `TokenBlocklistEntry(tokenId = jwtTokenProvider.getTokenId(accessToken), expiresAt = jwtTokenProvider.getExpiration(accessToken).toInstant())`
  2. If refreshToken is not null and valid, add to blocklist similarly
  3. Save both entries

- Define private `logger` using `LoggerFactory.getLogger(AuthService::class.java)`
- Define custom exceptions as inner classes or separate file: `class RegistrationException(message: String) : RuntimeException(message)` and `class AuthenticationException(message: String) : RuntimeException(message)` (use a custom name to avoid clashing with Spring's `AuthenticationException`)

**AuthController.kt** (in controller/ package):
- `@RestController @RequestMapping("/api/auth")` class, constructor-inject `AuthService`
- `@PostMapping("/register") fun register(@Valid @RequestBody request: RegisterRequest): ResponseEntity<MessageResponse>`:
  - Call `authService.register(request)`
  - Return `ResponseEntity.status(HttpStatus.CREATED).body(result)`

- `@PostMapping("/login") fun login(@Valid @RequestBody request: AuthRequest, response: HttpServletResponse): ResponseEntity<AuthResponse>`:
  - Call `authService.login(request)` to get `(authResponse, refreshToken)`
  - Create refresh cookie: `Cookie("refresh_token", refreshToken)` with:
    - `isHttpOnly = true`
    - `secure = false` (local dev; set true in production)
    - `path = "/api/auth/refresh"`
    - `maxAge = (jwtTokenProvider.getRefreshExpirationMs() / 1000).toInt()`
    - `setAttribute("SameSite", "Lax")`
  - Add cookie to response: `response.addCookie(cookie)`
  - Return `ResponseEntity.ok(authResponse)`

- `@PostMapping("/refresh") fun refresh(@CookieValue("refresh_token") refreshToken: String, response: HttpServletResponse): ResponseEntity<AuthResponse>`:
  - Call `authService.refresh(refreshToken)` to get new `(authResponse, newRefreshToken)`
  - Set new refresh cookie (same pattern as login)
  - Return `ResponseEntity.ok(authResponse)`

- `@PostMapping("/logout") fun logout(request: HttpServletRequest, response: HttpServletResponse): ResponseEntity<MessageResponse>`:
  - Extract access token from `Authorization` header (strip "Bearer ")
  - Extract refresh token from cookie `refresh_token` (may be null)
  - Call `authService.logout(accessToken, refreshToken)`
  - Clear the refresh cookie: create cookie with same name, `maxAge = 0`, `path = "/api/auth/refresh"`
  - Return `ResponseEntity.ok(MessageResponse("Logged out successfully"))`

- Inject `JwtTokenProvider` for cookie maxAge calculation
