## Interaction Contracts

### Timeline Tab

| State | Visual |
|-------|--------|
| Loading | 3 Skeleton blocks (h-16 w-full), stacked vertically with 16px gap, 16px top padding |
| Empty | "No timeline entries yet." in text-sm text-muted-foreground, 16px top padding |
| Populated | Vertical feed of entry cards, most-recent-first, 16px gap between entries |
| Error | TanStack Query default error -- no custom error UI (consistent with Notes/Interviews tabs) |

**Timeline Entry Card Layout:**

```
+--[border rounded-md p-3]----------------------------+
| [TypeIcon 16px]  [Type Badge text-xs]    [Date text-xs muted] |
| [Summary text-sm, full width]                        |
| [Details: key-value pairs, text-xs muted, if present]|
+------------------------------------------------------+
```

- Container: `div` with `border rounded-md p-3` (matches existing card patterns in detail panel)
- Row 1: Icon (16px, muted-foreground) + type badge + right-aligned relative date
- Row 2: Summary text, 14px body weight
- Row 3 (conditional): Details metadata displayed as key-value pairs if `metadata` is non-null. Always visible (no accordion). Text-xs muted-foreground.

**Timeline Type Icons (lucide-react):**

| Type | Icon | Rationale |
|------|------|-----------|
| INTERVIEW | `CalendarIcon` | Calendar event association |
| APPLICATION_NOTE | `StickyNoteIcon` | Note-taking metaphor |
| INTERVIEW_NOTE | `MessageSquareIcon` | Conversation details |

Source: CONTEXT.md Claude's Discretion + RESEARCH.md recommendation.

### Category Selector (Document Upload)

| State | Visual |
|-------|--------|
| Default | Select component showing "Other" (default value: OTHER) |
| Open | Dropdown with 4 options: CV, Cover Letter, Portfolio, Other |
| Selected | Shows selected label, value passed to formData on upload |

**Layout positioning:** Category selector placed ABOVE the dropzone area, with a Label "Category" and 8px gap between label and Select. Full width of the upload form container.

**Options (ordered):**

| Value | Label |
|-------|-------|
| CV | CV |
| COVER_LETTER | Cover Letter |
| PORTFOLIO | Portfolio |
| OTHER | Other |

**Behavior:** Category state persists across multiple file drops in the same session. One category applies to all files dropped in a single batch. State resets when the component unmounts (navigating away).

Source: CONTEXT.md locked decision (shadcn Select, default OTHER) + RESEARCH.md Fix 3.

### Interview Notes (Fix Only)

No visual changes. The hook layer unwraps `Page<InterviewNoteResponse>` to return `InterviewNoteResponse[]`. Component code remains unchanged.

### Route Guard (Verification Only)

No visual changes. Unauthenticated users accessing `/dashboard/*` routes are redirected to `/auth/sign-in`. This is a runtime verification of existing `proxy.ts`.
