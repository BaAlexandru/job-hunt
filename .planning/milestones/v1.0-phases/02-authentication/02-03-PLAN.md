**EmailVerificationService.kt** (in service/ package):
- `@Service` class, constructor-inject: `EmailVerificationTokenRepository`, `UserRepository`
- `fun verify(token: String): MessageResponse`:
  1. `val verificationToken = emailVerificationTokenRepository.findByToken(token) ?: throw InvalidTokenException("Invalid verification token")`
  2. `if (verificationToken.used) throw InvalidTokenException("Token already used")`
  3. `if (verificationToken.expiresAt.isBefore(Instant.now())) throw InvalidTokenException("Token expired")`
  4. Mark token used: `verificationToken.used = true`, save
  5. Enable user: `val user = verificationToken.user; user.enabled = true; user.updatedAt = Instant.now()`, save
  6. Return `MessageResponse("Email verified successfully. You can now log in.")`
- Define `class InvalidTokenException(message: String) : RuntimeException(message)`

**PasswordResetService.kt** (in service/ package):
- `@Service` class, constructor-inject: `UserRepository`, `PasswordResetTokenRepository`, `PasswordEncoder`, `RateLimiter`
- `private val logger = LoggerFactory.getLogger(PasswordResetService::class.java)`
- `fun requestReset(email: String): MessageResponse`:
  1. Rate limit check: `if (!rateLimiter.isAllowed("password-reset:$email", 3, 3600)) throw RateLimitException("Too many reset requests. Try again later.")`
  2. `val user = userRepository.findByEmail(email)` -- if null, still return success message (prevent email enumeration)
  3. If user exists: create token `UUID.randomUUID().toString()`, save `PasswordResetToken(user = user, token = token, expiresAt = Instant.now().plus(1, ChronoUnit.HOURS))`
  4. Log to console: `logger.info("Password reset link: http://localhost:8080/api/auth/password-reset/confirm?token=$token")`
  5. Always return: `MessageResponse("If an account with that email exists, a reset link has been sent.")`

- `fun confirmReset(token: String, newPassword: String): MessageResponse`:
  1. `val resetToken = passwordResetTokenRepository.findByToken(token) ?: throw InvalidTokenException("Invalid reset token")`
  2. `if (resetToken.used) throw InvalidTokenException("Token already used")`
  3. `if (resetToken.expiresAt.isBefore(Instant.now())) throw InvalidTokenException("Token expired")`
  4. Mark token used: `resetToken.used = true`, save
  5. Update password: `val user = resetToken.user; user.password = passwordEncoder.encode(newPassword); user.updatedAt = Instant.now()`, save user
  6. Return `MessageResponse("Password reset successfully. You can now log in with your new password.")`

- Define `class RateLimitException(message: String) : RuntimeException(message)`

**RateLimiter.kt** (in service/ package):
- `@Service` class, constructor-inject: `StringRedisTemplate`
- `fun isAllowed(key: String, maxRequests: Int, windowSeconds: Long): Boolean`:
  1. `val ops = redisTemplate.opsForValue()`
  2. `val current = ops.increment(key) ?: 1`
  3. `if (current == 1L) { redisTemplate.expire(key, Duration.ofSeconds(windowSeconds)) }`
  4. `return current <= maxRequests`
