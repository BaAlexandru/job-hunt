# JobHunt
https://job-hunt.dev/

Track jobs you've applied to with their status, documents, and timeline so nothing falls through the cracks during a job search.

## Tech Stack

- **Backend:** Spring Boot 4 + Kotlin (JDK 24)
- **Frontend:** Next.js 16 + React 19
- **Database:** PostgreSQL 17
- **Cache:** Redis 7
- **Storage:** MinIO (S3-compatible)
- **CI:** GitHub Actions -> GHCR
